## (Nameless) Namespaces

This app displays [Linux-kernel
namespaces](https://man7.org/linux/man-pages/man7/namespaces.7.html) discovered
by the **lxkns** service deployed to a Linux host. These
[namespaces](namespaces) partition certain kinds of Linux resources (such as the
file system, processes, network interfaces), so that processes that are members
of a particular namespace only see these resources isolated in that namespace.
**lxkns** is container-aware, helping with orientation.

In a weird twist of penguin logic, "namespaces" actually do not have any names.
Historically, the first type of namespace in Linux was the "mount" namespace
which is about system paths and partitioning mount points ... and this is where
the meaning of namespace actually originates from.

## Process/Task Scheduling and Affinity

Additionally, **lxkns** features showing the scheduling configuration of
processes and tasks and [how these can be executed on any or specific subsets of
online CPUs](/help/affinities).
